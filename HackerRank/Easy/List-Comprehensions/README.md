# List Comprehensions

## Problem Information

- **Platform:** HackerRank
- **Difficulty:** Easy
- **Problem Link:** https://www.hackerrank.com/challenges/list-comprehensions

---

## Problem Summary

Generate all possible coordinates `(i, j, k)` and exclude the coordinates where:

`i + j + k == n`

---

## Concept Learned

- Nested Loops
- Lists
- List Comprehensions

---

## My Approach

I solved this problem using three nested loops and stored the valid coordinates in a list.

Later, I will solve it again using Python List Comprehensions.
