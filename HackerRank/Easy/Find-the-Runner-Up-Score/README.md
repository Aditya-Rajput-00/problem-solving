# Find the Runner-Up Score!

## Problem Information

- **Platform:** HackerRank
- **Difficulty:** Easy

## Why I Created This README

This problem teaches how to find the second largest unique value from a list by removing duplicate values.

## Concepts Used

- List
- Set
- max()
- remove()

## My Approach

1. Convert the list to a set to remove duplicates.
2. Find the largest number.
3. Remove the largest number.
4. Find the largest number again.
5. Print the result.

## Time Complexity

O(n)

## Space Complexity

O(n)
