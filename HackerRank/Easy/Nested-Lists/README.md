## Why I Created This README

---

## Concepts Used

- Nested Lists
- Loops
- set()
- sorted()
- List Traversal
- String Sorting

---

## My Approach

1. Store all students and their grades.
2. Create a list of grades.
3. Remove duplicate grades using `set()`.
4. Sort the grades and find the second lowest grade.
5. Find all students with that grade.
6. Sort their names alphabetically.
7. Print the names.

---

## Time Complexity

O(n log n)

## Space Complexity
