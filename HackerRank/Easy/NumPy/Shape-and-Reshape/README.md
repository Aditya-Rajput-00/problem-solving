# Shape and Reshape

## Problem Information

- **Platform:** HackerRank
- **Difficulty:** Easy
- **Problem Link:** https://www.hackerrank.com/challenges/np-shape-reshape

---

## Why I Created This README

This is the first NumPy problem in the HackerRank Python track. It introduces array creation and reshaping, which are fundamental concepts in NumPy.

---

## Concepts Used

- NumPy
- np.array()
- np.reshape()

---

## My Approach

1. Read the input values.
2. Convert them into a NumPy array.
3. Reshape the array into a 3 × 3 matrix.
4. Print the reshaped array.
